### Overview
The Smart Student Analytics System (SSAS) is an AI-powered educational intelligence platform designed for Nigerian secondary schools. It provides real-time performance predictions, career recommendations, peer analysis, and anomaly detection using advanced machine learning models.

### üöÄ Quick Start Guide

#### 1. Accessing the System
- Open your web browser and navigate to the SSAS frontend
- You'll see a login screen with the SSAS branding

#### 2. Login
- **Username**: Your assigned username (e.g., `admin`)
- **Password**: Your secure password
- Click "Login" to access the dashboard

#### 3. Dashboard Overview
Once logged in, you'll see:
- **System Metrics**: Real-time statistics about students and predictions
- **Student Analytics**: Search and analyze individual student data
- **System Health**: Monitor the overall system status

### üìä Using Student Analytics

#### Step 1: Enter Student Information
1. In the "Student Analytics" section, enter a **Student ID** (e.g., `STD0001`)
2. Select a **Subject** from the dropdown menu:
   - **Critical Subjects**: Mathematics, English Language
   - **Science Subjects**: Physics, Chemistry, Biology, Agricultural Science
   - **Arts Subjects**: Literature, Government, Economics, History, Geography

#### Step 2: Get Predictions
Click the "Get Predictions" button to generate:
- **Performance Prediction**: Expected score for the selected subject
- **Career Recommendations**: Top career paths based on performance patterns
- **Peer Analysis**: Student's ranking within their peer group
- **Anomaly Detection**: Identification of unusual performance patterns

### üéØ Understanding the Results

#### Performance Prediction
- **Predicted Score**: The expected score (0-100) for the subject
- **Confidence Level**: How reliable the prediction is (0-1)
- **Subject**: The analyzed subject
- **Student ID**: The student being analyzed

#### Career Recommendations
- **Career Path**: Suggested career based on performance
- **Match Score**: How well the student fits this career (0-1)
- **Requirements**: Skills and subjects needed for this career

#### Peer Analysis
- **Peer Group**: Students in the same class/stream
- **Ranking**: Student's position within the peer group
- **Percentile**: Percentage of peers the student outperforms

#### Anomaly Detection
- **Anomalies**: Unusual patterns in student performance
- **Alerts**: Warnings about potential issues
- **Recommendations**: Suggested interventions
